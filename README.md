# 1. Compile

## FreeBSD, Linux

Debug mode.

```
$ cc fixed.c crypt.c main.c
$ ./a.out
```

Release mode.

```
$ cc -DFIXED_RELEASE fixed.c crypt.c main.c
$ ./a.out
```
